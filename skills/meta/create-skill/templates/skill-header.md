# Skill Header Template

Paste this line at the top of every new `SKILL.md` (right after the closing `---` of the frontmatter):

```markdown
*[Your Name](https://yourwebsite.com) · [𝕏](https://x.com/yourhandle) · [GitHub](https://github.com/yourusername) · [skills repo](https://github.com/yourusername/skills)*
```

---

**Minda's header (the default used by this skill):**

```markdown
*[Minda Myers](https://mindamyers.com) · [𝕏](https://x.com/MindaMyers) · [GitHub](https://github.com/Minda) · [skills repo](https://github.com/Minda/skills)*
```

Replace all five values — name, website, X handle, GitHub username, and skills repo URL — with your own before publishing.
