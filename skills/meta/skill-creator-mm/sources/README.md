# Sources for Skills

Curated external repositories, catalogs, and reference materials that skill authors should check **before** creating a new skill. Use these sources to find existing implementations, avoid duplication, and learn from canonical examples.

## Purpose

- **Discover existing skills** — Check if someone has already built what you need
- **Learn from implementations** — Study structure, frontmatter, and patterns
- **Avoid duplication** — Extend or fork instead of rebuilding from scratch

## Available Sources

| Source | Description |
|--------|-------------|
| `anthropic-skills.md` | Anthropic’s official skills repo — document, design, MCP, and more |
| `skills-catalog.json` | Local catalog of 200+ skills (at skill-creator root) |

When creating a new skill, scan the relevant source(s) first. Load the file when the use case matches that source’s domain.
